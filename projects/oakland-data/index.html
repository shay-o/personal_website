Oakland Open Data Chat

A chat interface to Oakland, California's open data portal (data.oaklandca.gov). Ask about crime, 311 service requests, street trees, campaign finance, voter turnout, and more.

This page demonstrates a third-party app consuming an MCP server: the Vercel function behind this chat connects to the Oakland Data MCP over streamable HTTP, discovers its tools dynamically, and runs an LLM tool-calling loop against OpenRouter. Source on GitHub →
