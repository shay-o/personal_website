Summary
Comparison
All four options solve the same problem — none require JavaScript changes, only CSS tweaks.

3: Indented Rows ★ REC Role items indented with vertical connector line ★★★★★ ★★★★★
4: Background Tiers Gray band on sections, white on role items ★★★★★ ★★★★★
Recommended: Option 3 (Indented Rows)

Indentation is the clearest structural metaphor for parent-child hierarchy — it's how file trees, outlines, and legal documents have worked for decades. It requires no change to section or role header styling (so the existing design language is preserved), and the connecting border-left line doubles as a visual affordance for the current section's contents. Options 2 and 4 also work well; Option 2 is a strong runner-up if you prefer to keep items full-width, since the size gap alone creates clear H2/H3 differentiation.
